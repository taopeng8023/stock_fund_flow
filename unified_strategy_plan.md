# 大秦统一选股框架 — 三周期全覆盖

## Context

用户需求：6万本金、5只持仓、仅主板(60/00)、目标赚钱。需要一套基于互联网公认策略的选股体系，覆盖短线/中线/长线三周期，输出策略文档+Python实现+回测验证。

---

## 策略设计：三层漏斗递进体系

核心理念：三层不是三个独立策略，而是**漏斗递进**——长线定"能不能买"，中线定"什么时候看"，短线定"什么时候动手"。

```
【第三层：长期质量过滤】F-Score + Magic Formula → quality_pool (~500只)
                    ↓
【第二层：中期动量择时】趋势+业绩缺口+渗透率+资金流 → momentum_pool (~50只)
                    ↓
【第一层：短期信号执行】现有隔夜信号 + SUE + 反包 → 最终5只买入
```

### 为什么是漏斗而非并行？
- 6万分3个并行策略各2万，仓位太薄，手续费侵蚀严重
- 漏斗让三层信号互相验证，降低单一信号误判
- 长线质量能过滤掉现有策略中最致命的垃圾股误判

## 三层策略规则

### 第三层：长期质量过滤（每季度更新）

**Piotroski F-Score (0-9分，≥6分通过)**

| # | 条件 |
|---|------|
| P1 | 净利润 > 0 |
| P2 | 经营现金流 > 0 |
| P3 | ROA 同比上升 |
| P4 | 经营现金流 > 净利润 |
| P5 | 长期负债率同比下降 |
| P6 | 流动比率同比上升 |
| P7 | 无增发新股 |
| P8 | 毛利率同比上升 |
| P9 | 资产周转率同比上升 |

**Magic Formula (百分位均值 ≥ 60%)**
- Earnings Yield = EBIT / Enterprise Value
- ROIC = (净利润-分红) / (总资产-流动负债)

通过条件: F-Score ≥ 6 AND Magic Formula百分位均值 ≥ 60%

### 第二层：中期动量择时（每日更新）

4维度评分，≥0.6进入momentum_pool:

| 维度 | 权重 | 指标 |
|------|------|------|
| 趋势确认 | 30% | MA多头排列层级、MA200上方+斜率 |
| 业绩缺口 | 25% | QoQ营收/利润增速、公告后5日涨幅、缺口支撑 |
| 板块渗透率 | 20% | 行业内正收益占比，5%-35%最佳 |
| 资金流累积 | 25% | 5/10/20日主力净流入、日均换手率 |

### 第一层：短期信号（盘中每30分钟更新）

复用现有信号体系(P0-P4/S1-S6) + 新增:

**B1: SUE信号**
```
SUE = (当前季度EPS - 4季度前EPS) / std(EPS变化, 过去8季度)
买入: SUE > 1.0
```

**B2: 反包信号**
```
条件: 前日缩量跌(-2%以上) + 当日放量涨(+2%以上) + 收盘价盖过前日开盘价 + 行业强度Top5
```

## 仓位分配

| 市场体制 | 总仓位 | 底仓(中长线) | 轮动仓(短线) | 现金 |
|---------|-------|------------|------------|------|
| bull | 80% | 40%(2只) | 40%(2只) | 20% |
| bull_bias | 50% | 30%(2只) | 20%(1只) | 50% |
| range | 35% | 20%(1只) | 15%(1只) | 65% |
| bear_bias | 25% | 15%(1只) | 10%(1只) | 75% |
| bear | 15% | 15%(1只) | 0% | 85% |

单只上限: 20%总权益(1.2万)，下限: 5%(0.3万)

### 跨时间框架一致性
- 长线F-Score < 4 但短线信号触发 → 降级为观察
- 中线动量 < 0.3 但短线信号触发 → 仓位减半
- 三层全部共振 → 仓位可提升1.5倍(不超上限)

## 卖出规则(10条)

复用现有7条 + 新增3条:

- **TE-3: 质量恶化** — F-Score下降≥2分 → 10天内清仓
- **TE-4: 板块退潮** — 行业渗透率跌破5%或超40% → 清仓该行业
- **TE-5: 信号反转** — SUE从正转负且abs>1 → 清仓

差异化止损:

| 周期 | 止损 | 止盈 | 最大持有天数 | 移动止盈回撤 |
|------|------|------|------------|------------|
| 短线 | -3% | +8% | 5 | -4% |
| 中线 | -8% | +15% | 20 | -8% |
| 长线 | -15% | +30% | 120 | -12% |

1. **因子有效性**: IC分析，分层收益单调性检验

## 关键风险

| 风险 | 应对 |
|------|------|
| 财报数据API不稳定 | 缓存45天，过期降权标记"stale" |
| F-Score在A股区分度不足 | 可降级为弱信号(0.5x权重)或仅做负向排除 |
| 6万手续费侵蚀 | 单只最低0.6万投入，控制月换手<50% |
| 季报发布时间滞后 | 增量采集模式，公告日匹配 |
