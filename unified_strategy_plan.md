# 大秦统一选股框架 — 三周期三层漏斗递进体系

## 一、理论基础：不同时间尺度的 alpha 来源完全不同

格雷厄姆在《聪明的投资者》中说："市场短期是一台投票机，长期是一台称重机"。这句话背后是现代金融学对三个时间尺度的 alpha 本质的完整揭示：

### 1.1 短线（天~周）：行为偏差套利

| 策略 | 学术来源 | 行为偏差 | 有效窗口 |
|------|---------|---------|---------|
| **SUE 预期差漂移** | Bernard & Thomas (1989)，Zacks《151 Trading Strategies》，18年回测 CAGR 13.6%/WR 65% | **反应不足**（underreaction）——市场对财报信息消化缓慢，超额指引期约 30-35 个交易日 | 季报后 1-30 天 |
| **反包/反转** | Kahneman & Tversky 前景理论（1979），马克·米勒维尼《股票魔法师》VCP 收缩 | **过度反应**（overreaction）——散户对利空消息过度恐慌抛售，聪明钱次日放量收复 | 1-3 天 |
| **涨停回调** | 招商证券 A 股博弈框架，A 股投资者二元结构（散户占比 ~60%） | **追涨杀跌**——追涨停后恐慌止损形成的超跌反弹 | 3-10 天 |

**短线本质**：赚别人犯错的钱。你不是在预测价格走向，你是在利用市场的系统性行为偏差。

### 1.2 中线（周~月）：景气趋势跟踪

| 策略 | 学术/实务来源 | 驱动逻辑 | 有效窗口 |
|------|-------------|---------|---------|
| **业绩断层** | 马克·米勒维尼 SEPA（Specific Entry Point Analysis），《股票魔法师》I/II | 季报超预期 → 机构重新定价 → 缺口不补就是强势确认。缺口本质是认知差的痕迹。 | 季报后 1-3 月 |
| **趋势动量** | 理查德·德里豪斯，"Buy high, sell higher"，Jegadeesh & Titman (1993) 动量因子 | 价格本身就是最诚实的基本面信号。强者恒强的根源是盈利动量尚未被充分定价。 | 3-12 月 |
| **产业渗透率** | 费雪《怎样选择成长股》，国投/招商产业赛道框架 | 渗透率 5%→35% 是产业加速期，个股跟随板块 β 最确定的一段 | 1-3 年 |

**中线本质**：赚趋势的钱。企业基本面在改善，同时市场正在发现这个改善过程——戴维斯双击（盈利↑ + 估值↑）。

### 1.3 长线（月~年）：质量价值回归

| 策略 | 学术/实务来源 | 驱动逻辑 | 有效窗口 |
|------|-------------|---------|---------|
| **F-Score 九因子** | Piotroski (2000, JAR)，《151 Trading Strategies》验证 WR 65% | 9 个简单会计指标区分"真正的困境反转"和"价值陷阱"。高分股年化超额 ~7%。 | 1-2 年 |
| **Magic Formula** | 格林布拉特《股市稳赚》（The Little Book That Beats the Market），1988-2004 回测 CAGR 30.8% | 好公司（高 ROIC）+ 便宜价格（高 EY）= 买到印钞机。每 3 个月换仓一次。 | 3-6 月持仓周期 |
| **质量复利** | 芒格《穷查理宝典》+ 费雪 + 巴菲特 | ROIC>15%、低杠杆、充裕自由现金流、有护城河。复利会替你工作。 | 3-5 年+ |

**长线本质**：赚企业创造价值的钱。买好公司 + 好价格 + 长期持有 = 让时间和复利成为你的盟友。

## 二、三层漏斗架构：不是选一个，是串起来

### 2.1 为什么要漏斗递进而不是三个独立策略

| 方案 | 优点 | 致命缺陷 |
|------|------|---------|
| 三个独立策略并行 | 简单 | 6 万分 3 个各 2 万，仓位太薄，单笔佣金 5 元（买入+卖出 10 元），0.5% 的手续费侵蚀不是玩的 |
| 漏斗递进体系 | 信号互相验证，质量有保障，仓位集中 | 长线标的可能暂时没有短线信号——这是正常的，长线仓允许等待 |

### 2.2 漏斗递进逻辑

```
全市场（主板 ~3000 只）
        │
        ▼  【第三层：长期质量过滤】F-Score ≥ 6 + Magic Formula 双因子均值 ≥ 60%
        │   ——格雷厄姆的安全边际 + 格林布拉特的"好公司便宜价"
        │   ——过滤掉注定会跌的垃圾公司（A 股亏钱头号原因）
   quality_pool (~300-500 只)
        │
        ▼  【第二层：中期动量择时】SEPA 趋势确认 + 业绩缺口 + 渗透率 + 资金流四维评分 ≥ 0.6
        │   ——米勒维尼的 SEPA + 德里豪斯的动量 + 费雪的产业加速期
        │   ——在好公司中，选出"正在变好"和"正在被发现"的那一批
   momentum_pool (~30-50 只)
        │
        ▼  【第一层：短期信号开火】SUE 预期差 + 反包 + 现有隔夜体系共振
        │   ——Bernard & Thomas 的反应不足 + Kahneman 的过度反应
        │   ——在正在变好的公司中，等到"别人犯错"的那个时刻开火
   最终持仓 3-5 只
```

### 2.3 三层各自的角色

| 层级 | 回答的问题 | 决策权 | 更新频率 |
|------|-----------|--------|---------|
| 第三层（质量） | 这只股票值不值得看？ | 一票否决：F-Score < 4 直接排除 | 每季报后 |
| 第二层（动量） | 它的好正在被市场发现吗？ | 仓位调节：动量 < 0.3 仓位减半 | 每日 |
| 第一层（信号） | 现在是不是最佳动手时刻？ | 触发买入：没有短线信号就不动 | 盘中每 30 分钟 |

### 2.4 三层共振的信号等级（借鉴米勒维尼的"多重确认"）

| 共振等级 | 条件 | 仓位动作 | 对应米勒维尼的 |
|---------|------|---------|-------------|
| **钻石** | 长+中+短 三者全部共振 | 仓位 ×1.5（不超 20% 单只上限） | "所有因子同时指向同一方向" |
| **黄金** | 长+中 共振，短等待 | 半仓先建，等短线信号补满 | "领先买入"（anticipation） |
| **白银** | 中+短 共振，长一般 | 正常仓位 | "趋势交易" |
| **观察** | 只有短线信号（长 F-Score < 4） | 不买，入观察列表 | 被否决——垃圾股反弹不值得参与 |
| **跳过** | 只有中长期，无短线信号 | 不买 | 好公司但时机不对 |

> 从 13 部经典投资文献提取，四层 11 因子，权重 100%。每项因子得分 0-1，加权汇总得最终排名。

### 3.3 逐因子详表

#### 质量层（权重 35%）——这公司好吗？

**F1: Piotroski F-Score（权重 15%，来源：Piotroski 2000, JAR）**

皮奥特罗斯基的洞见：高 B/M 公司的收益绝大部分来自少数真正反转的股票，大部分低价股是价值陷阱。F-Score 就是用来区分二者的。

```
F1 = (P1 + P2 + ... + P9) / 9（归一化到 0-1）
```

| # | 条件 | 类别 | 格雷厄姆/费雪对应 |
|---|------|------|-----------------|
| P1 | 净利润 > 0 | 盈利 | "公司必须盈利" |
| P2 | 经营现金流 > 0 | 盈利 | "现金流不会说谎" |
| P3 | ROA 同比上升 | 盈利 | "管理层能力" |
| P4 | 经营现金流 > 净利润 | 盈利 | "盈利质量"——应收账款不过大 |
| P5 | 长期负债率同比下降 | 杠杆 | 安全边际——杠杆减小 |
| P6 | 流动比率同比上升 | 杠杆 | 安全边际——流动性改善 |
| P7 | 无增发新股 | 杠杆 | 股东利益对齐——不被稀释 |
| P8 | 毛利率同比上升 | 运营 | 定价权/护城河 |
| P9 | 资产周转率同比上升 | 运营 | 运营卓越——用同样资产创造更多收入 |

```
得分映射:  F-Score ≥ 8 → 1.0   卓越反转
           7 → 0.85
           6 → 0.7
           5 → 0.5
           4 → 0.3
           ≤ 3 → 自动排除（不进入评分）
```

---

**F2: ROIC 资本回报率（权重 12%，来源：格林布拉特 + 巴菲特）**

格林布拉特：**ROIC 是预测企业长期回报的单一最强指标**。高 ROIC 意味着公司有真正的护城河——竞争对手砸钱也抢不走它的利润。

```
ROIC ≈ (净利润 + 利息费用) / (总资产 - 流动负债)

得分映射:  ROIC ≥ 25%  → 1.0   印钞机级别（茅台、片仔癀）
           15-25%     → 0.8   优秀生意
           10-15%     → 0.6   不错
           5-10%      → 0.3   一般
           < 5%       → 0.0   辛苦钱
```

---

**F3: 应计利润质量（权重 8%，来源：格雷厄姆 + Sloan 1996）**

格雷厄姆的洞见：**利润可以被会计手段操纵，现金流不会**。高应计 = 利润可能有人为放大。

```
应计比率 = (净利润 - 经营现金流) / 总资产

得分映射（越低越好）:
  < -8%  → 1.0   现金流远超利润（极度保守，最可信）
  -8~-2% → 0.8   现金流充裕
  -2~3%  → 0.6   正常范围
  3~8%   → 0.3   应计偏高，利润可能有水分
  > 8%   → 0.0   利润很可能造假
```

---

#### 价值层（权重 20%）——价格便宜吗？

**F4: 盈利收益率 EY（权重 12%，来源：格林布拉特）**

```
EY = EBIT / Enterprise Value
   = (净利润 + 利息费用 + 所得税) / (总市值 + 总负债 - 现金)

行业内百分位排名 → 线性映射到 0-1
```

格林布拉特：用 EBIT/EV 而非 P/E，因为 P/E 被不同杠杆水平和税率污染。EV 扣除了现金——账上躺着 100 亿现金和负债 100 亿的公司，市值相同但价值完全不同。

---

**F5: PEG 比率（权重 8%，来源：彼得·林奇）**

林奇法则：一家好公司的合理 PEG 应该约等于 1。PEG < 0.5 可能捡到了宝——但要确认不是价值陷阱。

```
PEG = PE_TTM / 近三年净利润复合增速(%)

得分映射:  PEG < 0.5  → 1.0   极度低估
           0.5-1.0   → 0.8   合理偏低估
           1.0-1.5   → 0.6   公允
           1.5-2.0   → 0.3   偏贵
           > 2.0     → 0.0   太贵了
           (净利润增速 ≤ 0 → 0.0)
```

---

#### 动量层（权重 30%）——时机对吗？

**F6: 米勒维尼四阶段识别（权重 10%，来源：《股票魔法师》）**

米勒维尼核心框架：股票生命周期分四个阶段，**只能在第二阶段（上升期）参与**。

```
阶段判定:
  第二阶段（上升期）:
    MA50 > MA150 > MA200, 价格 > MA50, MA200 斜率向上
    → 得分 1.0  ★ 唯一值得重仓的阶段

  第一阶段末期（筑底即将突破）:
    价格从 MA200 下方回升，距 MA200 < 5%
    → 得分 0.6

  第一阶段（筑底中）:
    价格围绕 MA200 震荡
    → 得分 0.4

  第三阶段（顶部）:
    波动放大，放量滞涨
    → 得分 0.2

  第四阶段（下跌）:
    价格 < MA200, MA200 斜率向下
    → 得分 0.0  一票否决
```

---

**F7: 德里豪斯 52 周动量（权重 8%，来源：德里豪斯）**

德里豪斯的第一条铁律："只买创 52 周新高的股票"。

```
价格位置 = 当前价 / 250 日最高价

得分映射:  ≥ 95%  → 1.0   接近创新高（德里豪斯理想区）
           85-95% → 0.8
           70-85% → 0.5
           50-70% → 0.3
           < 50%  → 0.0   腰斩股不碰（"底部最贵"）
```

---

**F8: VCP 波动率收缩（权重 6%，来源：《股票魔法师》）**

米勒维尼发现，每一波上涨之间的回调，波动率会逐步收缩——**这是筹码在"有信心的手"中集中的过程**，是机构悄悄建仓的痕迹。

```
量化 VCP 三个条件：

1. 最近回调幅度 < 前次回调幅度（收缩）
2. 回调期间量 < 前 20 日均量的 70%（缩量）
3. 价格在 MA20 上方 3% 以内（接近支撑但不破）

→ 三者全部满足 → 1.0（标准 VCP）
→ 满足 2 项   → 0.5
→ 其他        → 0.0
```

---

**F9: 板块渗透率（权重 6%，来源：费雪 + 招商证券）**

费雪《怎样选择成长股》：最好的投资时机是产业从 5% 向 35% 渗透的阶段——竞争格局未恶化，市场空间加速打开。

```
渗透率 = 行业内近 60 日正收益股票数 / 行业总股票数

得分映射:  15-35%  → 1.0   加速期，费雪的甜蜜区
           5-15%   → 0.8   加速初期
           35-60%  → 0.3   成熟期
           < 5%    → 0.2   导入期太早
           > 60%   → 0.1   饱和/衰退
```

---

#### 信号层（权重 15%）——可以开火了吗？

**F10: SUE 标准化未预期盈利（权重 8%，来源：Bernard & Thomas 1989, JAR）**

学术金融学最稳健异象之一：季报超预期后股价持续漂移 30-60 天。心理学根源是"锚定效应"——投资者对新信息调整不足。

```
SUE = (EPS_t - EPS_t-4Q) / std(EPS 季度变化, 过去 8 季度)

得分映射:  SUE > 3.0   → 1.0   巨大惊喜
           2.0 - 3.0  → 0.8   重大惊喜
           1.0 - 2.0  → 0.6   明确超预期
           0.5 - 1.0  → 0.4   轻微超预期
           0 - 0.5    → 0.2   勉强达标
           < 0        → 0.0   不及预期

特殊规则:
  - 季报公告日距今 > 30 天 → 信号归零（漂移窗口已过）
  - 仅在门禁层通过 + quality_pool 内有效
```

---

**F11: A 股主力资金累积（权重 7%，来源：A 股流动性研究）**

```
资金得分 = 0.4 × 5日主力净流全市场百分位
         + 0.3 × 10日主力净流全市场百分位
         + 0.3 × 20日主力净流全市场百分位

百分位直接作为 0-1 得分
```

### 3.4 综合评分公式

```
                        ┌──────────────────────────────────────┐
                        │  最终得分 = Σ(Fi × Wi)，i = 1..11   │
                        │                                      │
                        │  = F1×0.15 + F2×0.12 + F3×0.08     │  ← 质量 35%
                        │  + F4×0.12 + F5×0.08                │  ← 价值 20%
                        │  + F6×0.10 + F7×0.08                │  ← 动量 30%
                        │  + F8×0.06 + F9×0.06                │
                        │  + F10×0.08 + F11×0.07              │  ← 信号 15%
                        └──────────────────────────────────────┘
```

**选股流程**：
1. 全市场主板 → 门禁层 6 道过滤 → 约 1500 只
2. 11 因子逐一打分 → 综合排序
3. 行业分散约束（同行业最多 2 只）→ 取前 5 只
4. 综合得分 ≥ 65 才买入（低于此 = 不买）

**仓位系数**：
```
综合得分 ≥ 85 → signal_multiplier = 1.5x
综合得分 ≥ 75 → signal_multiplier = 1.2x
综合得分 ≥ 65 → signal_multiplier = 1.0x
综合得分 < 65 → 不买
```

| | F11 资金流 | 7% | A 股研究 | fund_flow | ✅ 已有 |

## 六、仓位分配——凯利公式 + 体制自适应

### 6.1 理论基础

凯利公式：最优仓位 = (盈亏比 × 胜率 - (1 - 胜率)) / 盈亏比

在 A 股场景下不可直接用（因为：胜率和盈亏比不确定、交易非独立、存在极端尾部风险），但其思想可用于仓位框架设计：**高把握 → 大仓位，低把握 → 小仓位**。

### 6.2 体制自适应仓位表

| 市场体制 | 总仓位上限 | 底仓(中长线) | 轮动仓(短线) | 现金 | 理由 |
|---------|-----------|------------|------------|------|------|
| bull | 80% | 40%（2 只） | 40%（2 只） | 20% | 米勒维尼：牛市可以激进 |
| bull_bias | 50% | 30%（2 只） | 20%（1 只） | 50% | 适度参与 |
| range | 35% | 20%（1 只） | 15%（1 只） | 65% | 震荡市太容易两边打脸 |
| bear_bias | 25% | 15%（1 只） | 10%（1 只） | 75% | 防守为主 |
| bear | 15% | 15%（1 只） | 0% | 85% | 格雷厄姆：熊市不必在场 |

### 6.3 单只仓位约束

| 参数 | 值 | 说明 |
|------|-----|------|
| 单只最大仓位 | 20% 总权益（≈1.2 万 @ 6 万） | 超过这个比例，任何单一黑天鹅都会重伤账户 |
| 单只最小仓位 | 5% 总权益（≈0.3 万） | 低于这个值，佣金 5 元占 0.33%，加上滑点不划算 |
| 行业集中度上限 | 同行业最多 2 只 | 避免行业黑天鹅重创整个组合 |

### 6.4 仓位递进规则（米勒维尼的"试探+加仓"模型）

米勒维尼从不一开始就上满仓。他的方法是：先小仓位试探，如果市场"证明你是对的"，再加仓。

```
初始仓位 = min(20% 总权益, 动态最大仓位 × signal_multiplier)

增仓条件（持有期间）：
  1. 持仓盈利 >= 5%（马开始跑了）
  2. 信号层级仍在（基本面和技术面都没有恶化）
  3. 大盘体制没有降级
  → 加仓至初始的 1.25 倍

减仓条件（持有期间）：
  1. 持仓亏损 >= 3%（跑错了方向）
  → 减仓至初始的 0.5 倍

清仓条件：见第七章卖出规则
```

## 七、卖出规则——10 条分层退出

### 7.1 为什么要卖出规则

格林布拉特在 Magic Formula 中的做法是：**每个季度末机械调仓**。芒格说："如果买的时候是对的，那就不应该卖"。但在 A 股场景下——T+1 结算、±10% 涨跌停、政策风险——你需要一套分层退出规则来保护本金。

卖出规则的层级：（越靠前越硬性，不可争议）

| 层级 | 类型 | 触发后必须执行 |
|------|------|-------------|
| SL（Stoploss） | 止损——保护本金 | 是，立即 |
| TP（Take Profit） | 止盈——保护利润 | 是，立即 |
| TE（Time Exit） | 时间退出——死钱/质量问题 | 是，但有执行窗口 |

### 7.2 现有 7 条规则（复用 portfolio/manager.py）

| 编号 | 类型 | 条件 | 适用周期 |
|------|------|------|---------|
| SL-1 | 硬止损 | 亏损 >= 止损% | 全部 |
| SL-2 | 移动止盈回撤 | 从持仓期间最高价回撤 >= 8% | 全部 |
| TP-1 | 目标止盈 | 盈利 >= 止盈% | 全部 |
| TP-2 | 快盈锁仓 | 持有 ≤ 3 天且涨幅 ≥ 8% | 短线 |
| TE-1 | 死钱退出 | 持有 ≥ 10 天，涨跌 ±2% 以内 | 中线 |
| TE-2 | 水下过久 | 持有 ≥ 5 天，亏损 ≥ 2% | 短线/中线 |
| OT-1 | 隔夜超时 | 时间 > 次日 10:15（隔夜策略专用） | 短线 |

### 7.3 新增 3 条规则（长线/中线专用）

| 编号 | 类型 | 条件 | 理论来源 | 适用周期 |
|------|------|------|---------|---------|
| **TE-3** | 质量恶化 | 新季报发布后 F-Score 下降 ≥ 2 分 | 皮奥特罗斯基：F-Score 恶化是公司基本面变差的最可靠信号 | 长线 |
| **TE-4** | 板块退潮 | 行业渗透率跌破 5% 或超过 40% | 费雪：产业过了加速期，继续持有的性价比大幅降低 | 中线/长线 |
| **TE-5** | 信号反转 | SUE 从正值转为负值且绝对值 > 1 | Bernard & Thomas：负 SUE 意味着盈利正在恶化 | 短线/中线 |

### 7.4 按时间周期差异化参数

| 周期 | 止损 | 止盈 | 最大持有天 | 移动止盈激活 | 移动止盈回撤 |
|------|------|------|----------|------------|------------|
| 短线（隔夜-5 天） | -3% | +8% | 5 | +5% | -4% |
| 中线（1-4 周） | -8% | +15% | 20 | +10% | -8% |
| 长线（1-6 月） | -15% | +30% | 120 | +15% | -12% |

**为什么要差异化？**

短线赚的是行为偏差的快速回归——退得慢了利润就没了。
长线赚的是企业价值的逐步实现——退得太快等于自己砍掉复利。
德雷曼在《逆向投资策略》中指出：过早止盈是个人投资者跑输机构的第一大原因。

## 八、风险管理——五层防护罩

### 8.1 系统级（复用现有）

| 层级 | 机制 | 触发条件 | 动作 |
|------|------|---------|------|
| 1 层 | 熔断 | 单日全市场跌幅（现有 risk_controller.py） | L1 加仓位/filter → L2 禁止买入 → L3 强制减仓 |
| 2 层 | 黑天鹅 | 13 条规则 Level 0-3（black_swan.py） | Level ≥ 2 禁止所有买入 |
| 3 层 | 日内中位数 | 全市资金流入中位数分层（现有） | Level 0-4 控制总仓位 |
| 4 层 | 行业分散 | 同行业最大 2 只 | 自动在漏斗中排除同行业第 3 只 |
| 5 层 | 崩溃闸门 | Gate1/Gate2/Gate3（buy_engine.py） | 单日跌幅/量比/市值阈值 |

### 8.2 策略级（新增：跨时间框架一致性检查）

```
买入前强制检查：

1. 长线否决：如果 F-Score < 4 → 即使短线信号再强也不买
   （格雷厄姆：在垃圾堆里捡便宜货不是投资，是投机）

2. 中线降级：如果 momentum_score < 0.3 → 短线信号触发后仓位减半
   （米勒维尼：不要在第四阶段的股票中做短线，胜率极低）

3. 体制降级：如果黑天鹅 Level ≥ 2 或熔断 L2+ → 所有买入取消
   （格雷厄姆：牛市是散户亏钱的主要原因——因为牛市让人忘记风险）
```

## 十、A 股特殊约束处理

| 约束 | 影响 | 处理方式 |
|------|------|---------|
| T+1 结算 | 买入当日不能卖，隔夜有风险 | 短线仓位买入前过 TE/OT/体制检查三层风控 |
| ±10% 涨跌停（主板） | 涨停买不到，跌停卖不掉 | 涨停当日不追买（FUNDAMENTAL_RULES 第 1 条） |
| 6 万本金 | 佣金侵蚀不可忽视 | 单只最小 0.3 万（佣金占比 < 0.5%），月换手 < 50% |
| 最多 5 只持仓 | 集中度天然高 | 漏斗体系本身就是集中化的，5 只已足够 |
| 仅主板（60/00） | 标的池缩小 ~40% | 排除 300/688/8 开头，主板仍有 ~3000 只足够筛选 |
| 季报发布滞后 | F-Score 更新不及时 | 增量采集 + 公告日匹配，超过 45 天标记 "stale" 降权 |
