# 大秦统一选股框架 — 三周期三层漏斗递进体系

## 一、理论基础：不同时间尺度的 alpha 来源完全不同

格雷厄姆在《聪明的投资者》中说："市场短期是一台投票机，长期是一台称重机"。这句话背后是现代金融学对三个时间尺度的 alpha 本质的完整揭示：

### 1.1 短线（天~周）：行为偏差套利

| 策略 | 学术来源 | 行为偏差 | 有效窗口 |
|------|---------|---------|---------|
| **SUE 预期差漂移** | Bernard & Thomas (1989)，Zacks《151 Trading Strategies》，18年回测 CAGR 13.6%/WR 65% | **反应不足**（underreaction）——市场对财报信息消化缓慢，超额指引期约 30-35 个交易日 | 季报后 1-30 天 |
| **反包/反转** | Kahneman & Tversky 前景理论（1979），马克·米勒维尼《股票魔法师》VCP 收缩 | **过度反应**（overreaction）——散户对利空消息过度恐慌抛售，聪明钱次日放量收复 | 1-3 天 |
| **涨停回调** | 招商证券 A 股博弈框架，A 股投资者二元结构（散户占比 ~60%） | **追涨杀跌**——追涨停后恐慌止损形成的超跌反弹 | 3-10 天 |

**短线本质**：赚别人犯错的钱。你不是在预测价格走向，你是在利用市场的系统性行为偏差。

### 1.2 中线（周~月）：景气趋势跟踪

| 策略 | 学术/实务来源 | 驱动逻辑 | 有效窗口 |
|------|-------------|---------|---------|
| **业绩断层** | 马克·米勒维尼 SEPA（Specific Entry Point Analysis），《股票魔法师》I/II | 季报超预期 → 机构重新定价 → 缺口不补就是强势确认。缺口本质是认知差的痕迹。 | 季报后 1-3 月 |
| **趋势动量** | 理查德·德里豪斯，"Buy high, sell higher"，Jegadeesh & Titman (1993) 动量因子 | 价格本身就是最诚实的基本面信号。强者恒强的根源是盈利动量尚未被充分定价。 | 3-12 月 |
| **产业渗透率** | 费雪《怎样选择成长股》，国投/招商产业赛道框架 | 渗透率 5%→35% 是产业加速期，个股跟随板块 β 最确定的一段 | 1-3 年 |

**中线本质**：赚趋势的钱。企业基本面在改善，同时市场正在发现这个改善过程——戴维斯双击（盈利↑ + 估值↑）。

### 1.3 长线（月~年）：质量价值回归

| 策略 | 学术/实务来源 | 驱动逻辑 | 有效窗口 |
|------|-------------|---------|---------|
| **F-Score 九因子** | Piotroski (2000, JAR)，《151 Trading Strategies》验证 WR 65% | 9 个简单会计指标区分"真正的困境反转"和"价值陷阱"。高分股年化超额 ~7%。 | 1-2 年 |
| **Magic Formula** | 格林布拉特《股市稳赚》（The Little Book That Beats the Market），1988-2004 回测 CAGR 30.8% | 好公司（高 ROIC）+ 便宜价格（高 EY）= 买到印钞机。每 3 个月换仓一次。 | 3-6 月持仓周期 |
| **质量复利** | 芒格《穷查理宝典》+ 费雪 + 巴菲特 | ROIC>15%、低杠杆、充裕自由现金流、有护城河。复利会替你工作。 | 3-5 年+ |

**长线本质**：赚企业创造价值的钱。买好公司 + 好价格 + 长期持有 = 让时间和复利成为你的盟友。

## 二、三层漏斗架构：不是选一个，是串起来

### 2.1 为什么要漏斗递进而不是三个独立策略

| 方案 | 优点 | 致命缺陷 |
|------|------|---------|
| 三个独立策略并行 | 简单 | 6 万分 3 个各 2 万，仓位太薄，单笔佣金 5 元（买入+卖出 10 元），0.5% 的手续费侵蚀不是玩的 |
| 漏斗递进体系 | 信号互相验证，质量有保障，仓位集中 | 长线标的可能暂时没有短线信号——这是正常的，长线仓允许等待 |

### 2.2 漏斗递进逻辑

```
全市场（主板 ~3000 只）
        │
        ▼  【第三层：长期质量过滤】F-Score ≥ 6 + Magic Formula 双因子均值 ≥ 60%
        │   ——格雷厄姆的安全边际 + 格林布拉特的"好公司便宜价"
        │   ——过滤掉注定会跌的垃圾公司（A 股亏钱头号原因）
   quality_pool (~300-500 只)
        │
        ▼  【第二层：中期动量择时】SEPA 趋势确认 + 业绩缺口 + 渗透率 + 资金流四维评分 ≥ 0.6
        │   ——米勒维尼的 SEPA + 德里豪斯的动量 + 费雪的产业加速期
        │   ——在好公司中，选出"正在变好"和"正在被发现"的那一批
   momentum_pool (~30-50 只)
        │
        ▼  【第一层：短期信号开火】SUE 预期差 + 反包 + 现有隔夜体系共振
        │   ——Bernard & Thomas 的反应不足 + Kahneman 的过度反应
        │   ——在正在变好的公司中，等到"别人犯错"的那个时刻开火
   最终持仓 3-5 只
```

### 2.3 三层各自的角色

| 层级 | 回答的问题 | 决策权 | 更新频率 |
|------|-----------|--------|---------|
| 第三层（质量） | 这只股票值不值得看？ | 一票否决：F-Score < 4 直接排除 | 每季报后 |
| 第二层（动量） | 它的好正在被市场发现吗？ | 仓位调节：动量 < 0.3 仓位减半 | 每日 |
| 第一层（信号） | 现在是不是最佳动手时刻？ | 触发买入：没有短线信号就不动 | 盘中每 30 分钟 |

### 2.4 三层共振的信号等级（借鉴米勒维尼的"多重确认"）

| 共振等级 | 条件 | 仓位动作 | 对应米勒维尼的 |
|---------|------|---------|-------------|
| **钻石** | 长+中+短 三者全部共振 | 仓位 ×1.5（不超 20% 单只上限） | "所有因子同时指向同一方向" |
| **黄金** | 长+中 共振，短等待 | 半仓先建，等短线信号补满 | "领先买入"（anticipation） |
| **白银** | 中+短 共振，长一般 | 正常仓位 | "趋势交易" |
| **观察** | 只有短线信号（长 F-Score < 4） | 不买，入观察列表 | 被否决——垃圾股反弹不值得参与 |
| **跳过** | 只有中长期，无短线信号 | 不买 | 好公司但时机不对 |

### 3.3 逐因子详表

#### 质量层（权重 35%）——这公司好吗？

**F1: Piotroski F-Score（权重 15%，来源：Piotroski 2000, JAR）**

皮奥特罗斯基的洞见：高 B/M 公司的收益绝大部分来自少数真正反转的股票，大部分低价股是价值陷阱。F-Score 就是用来区分二者的。

```
F1 = (P1 + P2 + ... + P9) / 9（归一化到 0-1）
```

| # | 条件 | 类别 | 格雷厄姆/费雪对应 |
|---|------|------|-----------------|
| P1 | 净利润 > 0 | 盈利 | "公司必须盈利" |
| P2 | 经营现金流 > 0 | 盈利 | "现金流不会说谎" |
| P3 | ROA 同比上升 | 盈利 | "管理层能力" |
| P4 | 经营现金流 > 净利润 | 盈利 | "盈利质量"——应收账款不过大 |
| P5 | 长期负债率同比下降 | 杠杆 | 安全边际——杠杆减小 |
| P6 | 流动比率同比上升 | 杠杆 | 安全边际——流动性改善 |
| P7 | 无增发新股 | 杠杆 | 股东利益对齐——不被稀释 |
| P8 | 毛利率同比上升 | 运营 | 定价权/护城河 |
| P9 | 资产周转率同比上升 | 运营 | 运营卓越——用同样资产创造更多收入 |

```
得分映射:  F-Score ≥ 8 → 1.0   卓越反转
           7 → 0.85
           6 → 0.7
           5 → 0.5
           4 → 0.3
           ≤ 3 → 自动排除（不进入评分）
```

##### 格雷厄姆防御型投资者 7 准则（F1 追加项，满足 4/7 额外 +0.05）

格雷厄姆在《聪明的投资者》第 14 章给出了 7 条针对"防御型投资者"的硬性选股标准。满足 ≥ 4 条 = F1 额外加分 0.05（上限 1.0）。

| # | 准则 | 量化 | 对应 |
|---|------|------|------|
| G1 | 适当的企业规模 | 营收 > 10 亿（A 股工业股）/ 营收 > 5 亿（公用事业） | G4 升级版 |
| G2 | 强劲的财务状况 | 流动比率 ≥ 2.0 | P6 加强版 |
| G3 | 利润稳定性 | 近 5 年每年都有净利润 | P1 扩展到 5 年 |
| G4 | 股息记录 | 近 5 年连续派息 | 新增 |
| G5 | 利润增长 | 近 3 年 EPS 增长 ≥ 33%（年化 ~10%） | 新增 |
| G6 | 适度的 PE | PE_TTM < 15（或 PE < 过去 3 年平均利润 × 15 / 价格） | 估值保护 |
| G7 | 适度的 PB | PE_TTM × PB < 22.5（即格雷厄姆数字条件） | 双低保护 |

```
防御加分 = 满足 ≥ 4 条 → F1 = min(F1 + 0.05, 1.0)
```

**F2: ROIC 资本回报率（权重 12%，来源：格林布拉特 + 巴菲特）**

格林布拉特：**ROIC 是预测企业长期回报的单一最强指标**。高 ROIC 意味着公司有真正的护城河——竞争对手砸钱也抢不走它的利润。

```
ROIC ≈ (净利润 + 利息费用) / (总资产 - 流动负债)

得分映射:  ROIC ≥ 25%  → 1.0   印钞机级别（茅台、片仔癀）
           15-25%     → 0.8   优秀生意
           10-15%     → 0.6   不错
           5-10%      → 0.3   一般
           < 5%       → 0.0   辛苦钱
```

##### 巴菲特护城河五维度评分（来源：巴菲特致股东信 + 晨星护城河方法论）

巴菲特反复强调："我会买一个有护城河的公司，哪怕管理层平庸。我不会买一个没有护城河的公司，哪怕管理层是天才。" ROIC 高是护城河的**结果**，但护城河本身可以由以下五维度量化评估：

| 护城河维度 | 量化代理指标 | 满分条件 | 得分 |
|-----------|------------|---------|------|
| **定价权** | 5 年毛利率均值 > 行业均值 + 5% | 显著高于同行 | +0.2 |
| **转换成本** | 应收账款周转率低于行业均值（客户提前付款=依赖你） | 周转率 < 行业均值 × 0.7 | +0.2 |
| **网络效应** | 营收增速 > 行业均值 且 销售费用率 < 行业均值（不花钱推广就增长） | 双条件满足 | +0.2 |
| **规模优势** | 行业营收排名 Top 5 | 前五 | +0.2 |
| **品牌/牌照** | 毛利率 > 50%（消费品）或 持稀缺牌照（金融/医药） | 满足其一 | +0.2 |

```
护城河加分 = Σ(五维度得分)
护城河加分 > 0.5 → F2 额外 +0.05（上限 1.0）
护城河加分 = 0   → F2 不做调整
```

**F3: 应计利润质量（权重 8%，来源：格雷厄姆 + Sloan 1996 + 巴菲特）**

格雷厄姆的洞见：**利润可以被会计手段操纵，现金流不会**。高应计 = 利润可能有人为放大。巴菲特更进一步，提出"所有者盈余"（Owner Earnings）——比报告利润更真实地反映企业为股东创造的价值。

```
应计比率 = (净利润 - 经营现金流) / 总资产

得分映射（越低越好）:
  < -8%  → 1.0   现金流远超利润（极度保守，最可信）
  -8~-2% → 0.8   现金流充裕
  -2~3%  → 0.6   正常范围
  3~8%   → 0.3   应计偏高，利润可能有水分
  > 8%   → 0.0   利润很可能造假
```

##### 巴菲特所有者盈余调整（F3 子项）

巴菲特在 1986 年致股东信中指出：报告利润 + 折旧 - 维持性资本支出 = 真正的"所有者盈余"。本框架近似替代：

```
所有者盈余比率 = 经营现金流 / 净利润

如果 经营现金流 > 净利润（比率 > 1.0）:
  → F3 再加 0.1 分（上限 1.0）→ 盈利质量额外确认
如果 经营现金流 / 净利润 < 0.5:
  → F3 扣 0.1 分 → 盈利质量存疑
```

#### 价值层（权重 20%）——价格便宜吗？

**F4: 盈利收益率 EY（权重 12%，来源：格林布拉特）**

```
EY = EBIT / Enterprise Value
   = (净利润 + 利息费用 + 所得税) / (总市值 + 总负债 - 现金)

行业内百分位排名 → 线性映射到 0-1
```

格林布拉特：用 EBIT/EV 而非 P/E，因为 P/E 被不同杠杆水平和税率污染。EV 扣除了现金——账上躺着 100 亿现金和负债 100 亿的公司，市值相同但价值完全不同。

##### 格雷厄姆数字（F4 交叉验证）

格雷厄姆在《聪明的投资者》第 14 章给出了著名的估值上限公式：

```
格雷厄姆数字 = sqrt(22.5 × EPS_TTM × BVPS)

其中 22.5 = 最大 PE(15) × 最大 PB(1.5)

规则: 如果当前股价 > 格雷厄姆数字 → 超出格雷厄姆安全边际
      - 对 F4 附加扣分：价格 > 格雷厄姆数字 × 1.5 → F4 扣 0.1
      - 对 F4 附加加分：价格 < 格雷厄姆数字 × 0.7 → F4 加 0.05
```

---

**F5: PEG 比率（权重 8%，来源：彼得·林奇）**

林奇法则：一家好公司的合理 PEG 应该约等于 1。PEG < 0.5 可能捡到了宝——但要确认不是价值陷阱。但 PEG 不能一刀切——林奇的六类公司各有不同的估值逻辑。

##### 林奇六类公司分类（PEG 适用的前提条件）

| 类别 | 特征 | 合理 PEG | PEG 调整 | 林奇原话 |
|------|------|---------|---------|---------|
| **慢速增长** | >3% 增速，大盘蓝筹 | PEG 1.0-1.5 | 得分 ×1.0 | "买它们是为了股息" |
| **稳健增长** | 10-12% 增速 | PEG 0.8-1.2 | 得分 ×1.0 | "低买高卖，30-50% 就走" |
| **快速增长** | 20-25%+ 增速 | PEG < 1.0 | 得分 ×1.1 | "只要增长 > PE，就能赚钱" |
| **周期股** | 盈利大起大落 | PEG 失效！ | PEG = 1.0 | "PE 高时买入，PE 低时卖出"——反逻辑 |
| **困境反转** | 扭亏，关注存活能力 | PEG 失效！ | PEG = 0.5 | "关键是活不活得下来，不是估值" |
| **隐蔽资产** | 土地/品牌/专利/现金 | PEG 失效！ | PEG = 0.5 | "你买的是资产，不是利润" |

```
PEG = PE_TTM / 近三年净利润复合增速(%)

得分映射:  PEG < 0.5  → 1.0   极度低估
           0.5-1.0   → 0.8   合理偏低估
           1.0-1.5   → 0.6   公允
           1.5-2.0   → 0.3   偏贵
           > 2.0     → 0.0   太贵了
           (净利润增速 ≤ 0 → 0.0)

林奇特殊规则:
  - PEG 仅适用于"快速增长"和"稳健增长"类公司
  - 周期股不适用 PEG（盈利高峰时 PE 最低反而是卖出信号）
  - PEG 的"G"必须是可持续的增速，非一次性利润暴增
```

##### 林奇库存预警（反向信号）

林奇发现，**存货增速持续超过销售增速是最危险的信号之一**——意味着产品卖不出去、未来必减值。

```
库存预警 = (存货增速 - 营收增速) > 10%
→ 如果触发：F5 得分打八折（×0.8）
→ 如果库存增速 > 营收增速 > 20%：直接 F5 = 0（太危险）
```

#### 动量层（权重 30%）——时机对吗？

**F6: 米勒维尼趋势模板（权重 10%，来源：《股票魔法师》I/II）**

米勒维尼在《股票魔法师》中给出了精确的 8 条趋势模板条件——只有全部满足，才能确认股票处于"第二阶段（上升期）"。这是他的 SEPA 方法中最核心、最可量化的一套规则。

##### 米勒维尼 8 条趋势模板

| # | 条件 | 量化规则 | 来源章节 |
|---|------|---------|---------|
| T1 | 股价 > MA50 > MA150 > MA200 | MA50 ＞ MA150 ＞ MA200，且当前价 ＞ MA50 | 《魔法师I》P120 |
| T2 | MA150 > MA200 至少 1 个月 | 过去 20 个交易日 MA150 持续 > MA200 | "均线需要时间粘合向上" |
| T3 | MA200 已经上升至少 1 个月 | MA200 上倾（当前 > 20 日前），理想 4-5 个月 | "底部筑底需要时间" |
| T4 | MA50 位于 MA150 和 MA200 之上 | 已经形成明确的金叉排列 | "短期均线要领先长期" |
| T5 | 股价 ＞ 52 周最低价 至少 30% | 当前价 / 52 周最低价 ≥ 1.3 | "摆脱底部区域" |
| T6 | 股价在 52 周最高价 25% 以内 | 当前价 / 52 周最高价 ≥ 0.75 | "接近创新高是强势标志" |
| T7 | 相对强弱线（RS）在 52 周高点附近 | 个股涨幅/指数涨幅，52 周百分位 ≥ 80% | "比市场强才值得买" |
| T8 | 当前价 ＞ MA50 ＞ MA150 ＞ MA200 | 与 T1 重复确认，关键在于**梯度** | "均线应该像阶梯一样排列" |

```
F6 得分（米勒维尼 8 条趋势模板）:
  满足 8 条 → 1.0   标准第二阶段 ★ 最佳
  满足 6-7 条 → 0.8  接近第二阶段
  满足 4-5 条 → 0.5  过渡态
  满足 2-3 条 → 0.3  第一/第三阶段
  满足 ≤ 1 条 → 0.0  第四阶段 — 不参与
```

##### 米勒维尼的 D 阶段增强（提高 F6 精度的补充规则）

米勒维尼特别警告：**刚进入第二阶段的 4 周内是风险最高的时候**——趋势还不稳定。统计显示，进入第二阶段超过 4 周后再买入，胜率提升约 10%。

```
如果 T1-T8 全部满足 AND 首次满足距今 ≥ 20 个交易日:
  → F6 = 1.0（稳定第二阶段）
如果 T1-T8 全部满足 AND 首次满足距今 < 20 个交易日:
  → F6 = 0.85（刚进入，趋势未稳，密切关注）

**F8: VCP 波动率收缩（权重 6%，来源：《股票魔法师》）**

米勒维尼发现，每一波上涨之间的回调，波动率会逐步收缩——**这是筹码在"有信心的手"中集中的过程**，是机构悄悄建仓的痕迹。

#### 信号层（权重 15%）——可以开火了吗？

**F10: SUE 标准化未预期盈利（权重 8%，来源：Bernard & Thomas 1989, JAR）**

学术金融学最稳健异象之一：季报超预期后股价持续漂移 30-60 天。心理学根源是"锚定效应"——投资者对新信息调整不足。

得分映射:  SUE > 3.0   → 1.0   巨大惊喜
           2.0 - 3.0  → 0.8   重大惊喜
           1.0 - 2.0  → 0.6   明确超预期
           0.5 - 1.0  → 0.4   轻微超预期
           0 - 0.5    → 0.2   勉强达标
           < 0        → 0.0   不及预期

特殊规则:
  - 季报公告日距今 > 30 天 → 信号归零（漂移窗口已过）
  - 仅在门禁层通过 + quality_pool 内有效
```

### 3.7 权重设计的实证支撑（来源：詹姆斯·奥肖内西《华尔街股市投资经典》）

奥肖内西在 44 年（1963-2007）的跨周期回测中，系统比较了 50 多种因子的长期表现。以下是因子持久性排名及其对本框架权重的影响：

| 因子 | 44 年 CAGR | 持续性评级 | 奥肖内西结论 | 本框架对应 |
|------|-----------|-----------|------------|----------|
| **6/12 月价格动量** | 14.4% | ★★★★★ 极强 | "买入过去 6 个月表现最好的十分之一"——唯一在所有市场周期中都赚钱的策略 | F6 趋势 10% + F7 52 周动量 8% = **18%** |
| **低 P/S (<1.0)** | 14.1% | ★★★★ 强 | 低市销率 + 高动量是最强组合 | 间接体现在 F4 EY（EV 含市值） |
| **低 P/E (最低 10%)** | 13.5% | ★★★★ 强 | 低 PE 策略在所有期间都跑赢，但单独用波动太大 | 间接在 F5 PEG 中 |
| **低 P/B (最低 10%)** | 12.8% | ★★★ 中等 | 与 Piotroski 结合后效果大幅提升 | F1 F-Score 的原生适用域 |
| **高股息率** | 11.7% | ★★ 弱 | 单独使用不如直接选低 PE | 未独立纳入（格雷厄姆 G4 做加分项） |
| **ROE** | — | ★★ 弱 | 单独 ROE 并无持续超额收益 | F2 ROIC 替代（格林布拉特验证更强） |

**奥肖内西的终极配方**（与本框架一致）：
> "买入低 P/S + 高 6 月动量的 top 10%，再按低 PE 排除最贵的"— 这个组合策略在几乎所有回测周期都排第一。

对应本框架：**F4 EY + F5 PEG（价值 20%）× F6 F7（动量 18%）× F1 F-Score（质量 15%）**——三大因子群的权重总和 53%，正是奥肖内西验证最稳健的组合基石。

**奥肖内西的反直觉警告**：
> "在美股 1963-2007 的 44 年间，投资者如果坚持持有表现最差的策略（高 PE、高 PS、买跌），每 1 万美元最终变成 2.5 万美元；而持有最好的策略（低 PS、动量），最终变成 800 万美元。差距不是 10% 对 8%，而是一生一次的巨大鸿沟。"

### 3.8 德雷曼逆向价值补丁（来源：《逆向投资策略》）

大卫·德雷曼是行为金融学投资实践的鼻祖。他的核心发现：**市场系统性地高估了热门股、低估了冷门股**，因为分析师总在外推近期的好/坏消息。他的低 PE 策略在多个市场周期中都持续跑赢。

#### 德雷曼三低百分位（F4 辅助验证，不占独立权重，用于交叉确认）

```
低 PE 百分位: 全市场 PE_TTM 排名，取最低 20% 的股票 → +0.05 加到 F4
低 PB 百分位: 全市场 PB 排名，取最低 20% 的股票 → +0.05 加到 F4
低 PC 百分位: 全市场 P/CF 排名，取最低 20% 的股票 → +0.05 加到 F4

三低全部满足 → F4 额外加 0.15（上限 1.0）
三低全部不满足 → F4 不调整
```

德雷曼特别注意：**"低 PE 策略的成功不是因为低 PE 本身，而是因为它系统性地押注在了市场过度悲观的股票上。"** ——这就是为什么低 PE 注定是反人性的：你买入的永远是被大家讨厌的东西。

#### 德雷曼分析师过度反应效应（融入 F10 SUE 信号增强）

德雷曼发现，分析师一致预期在极端乐观和极端悲观时，系统性地跑偏。这意味着：

```
如果 SUE > 1.5 AND 近 3 个月分析师评级"卖出/减持"比例 > 30%:
  → 市场极度悲观，但公司实际超预期 → F10 额外 +0.1（"预期差更大"）

如果 SUE < -1.0 AND 近 3 个月分析师评级"买入/增持"比例 > 80%:
  → 市场极度乐观，但公司实际不及预期 → 触发 TE-5 卖出信号
```

### 3.9 格雷厄姆 NCAV（净流动资产）终极安全边际（来源：《证券分析》《聪明的投资者》）

格雷厄姆最激进的价值公式——买公司的流动变现价值。

```
NCAV = 流动资产 - 总负债
每股 NCAV = NCAV / 总股本

格雷厄姆买入标准: 股价 < 每股NCAV × 2/3
  — 这意味着你用 0.67 元买到了 1 元的流动资产，固定资产和持续经营能力全部白送

A 股适用性:
  - 极少有 A 股主板标的满足（通常只在极端熊市出现）
  - 如果满足 → 自动标记为"格雷厄姆特级价值"，直接跳过 F4/F5 进入质量层二次审核
  - 不满足 → 不做惩罚（太稀缺了，不能作为常规排除条件）
```

**格雷厄姆自己的统计**：在 30 年的实践中，NCAV 策略的年化回报超过 20%，最大回撤 < 30%。

## 六、仓位分配——凯利公式 + 体制自适应

### 6.1 理论基础

凯利公式：最优仓位 = (盈亏比 × 胜率 - (1 - 胜率)) / 盈亏比

在 A 股场景下不可直接用（因为：胜率和盈亏比不确定、交易非独立、存在极端尾部风险），但其思想可用于仓位框架设计：**高把握 → 大仓位，低把握 → 小仓位**。

### 6.2 体制自适应仓位表

| 市场体制 | 总仓位上限 | 底仓(中长线) | 轮动仓(短线) | 现金 | 理由 |
|---------|-----------|------------|------------|------|------|
| bull | 80% | 40%（2 只） | 40%（2 只） | 20% | 米勒维尼：牛市可以激进 |
| bull_bias | 50% | 30%（2 只） | 20%（1 只） | 50% | 适度参与 |
| range | 35% | 20%（1 只） | 15%（1 只） | 65% | 震荡市太容易两边打脸 |
| bear_bias | 25% | 15%（1 只） | 10%（1 只） | 75% | 防守为主 |
| bear | 15% | 15%（1 只） | 0% | 85% | 格雷厄姆：熊市不必在场 |

### 6.3 单只仓位约束

| 参数 | 值 | 说明 |
|------|-----|------|
| 单只最大仓位 | 20% 总权益（≈1.2 万 @ 6 万） | 超过这个比例，任何单一黑天鹅都会重伤账户 |
| 单只最小仓位 | 5% 总权益（≈0.3 万） | 低于这个值，佣金 5 元占 0.33%，加上滑点不划算 |
| 行业集中度上限 | 同行业最多 2 只 | 避免行业黑天鹅重创整个组合 |

### 6.4 仓位递进规则（米勒维尼的"试探+加仓"模型）

米勒维尼从不一开始就上满仓。他的方法是：先小仓位试探，如果市场"证明你是对的"，再加仓。

```
初始仓位 = min(20% 总权益, 动态最大仓位 × signal_multiplier)

增仓条件（持有期间）：
  1. 持仓盈利 >= 5%（马开始跑了）
  2. 信号层级仍在（基本面和技术面都没有恶化）
  3. 大盘体制没有降级
  → 加仓至初始的 1.25 倍

减仓条件（持有期间）：
  1. 持仓亏损 >= 3%（跑错了方向）
  → 减仓至初始的 0.5 倍

清仓条件：见第七章卖出规则
```

### 6.5 格林布拉特机械调仓规则（来源：《股市稳赚》）

格林布拉特的 Magic Formula 不需要判断"何时卖"——每个季度末机械地重新排名，卖出跌出排名的，买入新进入排名的。这种方法的最大优势是：**消除了情绪干扰和过度思考**。

```
季度调仓规则（长线持仓专用）:

1. 每季报发布后（约 1/5/9/11 月）：
   a. 对所有持仓重新跑一遍 11 因子评分
   b. 如果某只持仓综合得分 < 55 → 10 天内卖出（不给借口）
   c. 如果某只持仓综合得分 55-65 → 标记"观察"，下次调仓日还低就卖
   d. 如果买入候选综合得分 > 当前持仓得分 → 换仓

2. "不用脑子"规则（格林布拉特核心原则）：
   - 调仓日到了 → 必须做一次重新评估
   - 不要试图在"更好的价格"卖出——卖在调仓日
   - 不要因为"上次卖了就涨"而下次不卖——这是后见之明

3. 换手控制：
   - 单次调仓换手率 ≤ 50%（避免全换）
   - 每次最多替换 2 只（5 只持仓 × 50% ≈ 2.5，取 2）
```

### 6.6 格雷厄姆股债比价法则（来源：《聪明的投资者》第 8 章）

格雷厄姆建议防御型投资者根据股票收益率和债券收益率的相对高低，动态调整股债比例。这个指标至今仍是最被低估的市场风向标之一。

```
股债比价 = 全市场 EY / 10 年期国债收益率

格雷厄姆规则:
  股债比价 > 2.0   → 股票极度便宜 → 最高仓位 80%（bull 体制）
                   格雷厄姆：这是重仓股票的时机

  1.5 - 2.0        → 股票偏便宜   → 正常仓位

  1.0 - 1.5        → 股票估值合理   → 仓位降至 60%

  0.5 - 1.0        → 股票偏贵       → 仓位降至 40%
                   格雷厄姆：应增加债券/现金比例

  < 0.5            → 股票极度贵（EY < 国债收益率的一半）
                   → 仓位降至 25%，即使 bull 体制也不激进
                   → 格雷厄姆原话："这种情况历史上极少发生，但发生时必有大事"
```

**当前应用**：中国的 10 年期国债收益率约 2.5%。如果沪深 300 的 EY=(1/PE)≈6%，则比价 = 2.4，处于格雷厄姆"极度便宜"区间。

### 6.7 CAN SLIM 市场方向（来源：威廉·欧奈尔《笑傲股市》）

欧奈尔在 50 年实践中总结的 CAN SLIM 系统，是美国个人投资者协会（AAII）47 年实盘回测中年化收益排名第一的选股体系。其中最重要的不是选股（C-A-N-S-L-I）——而是 **M = Market Direction（市场方向）**。

> 欧奈尔："即使你选到世界上最好的股票，如果大盘处于下行趋势，3/4 的股票都会跟随下跌。不是你的选股能力变差了——是你在错误的时间游泳。"

#### 欧奈尔跟风日（Follow-Through Day, FTD）

```
1. 反弹尝试（Rally Attempt）:
   - 大盘指数录得低点后的第 1 天起开始计数
   - 第 1-3 天内出现单日涨幅 ≥ 1.7% 的上证综指 → 不算 FTD（太早，可能是假信号）
   - 必须在第 4 天及之后才考虑确认

2. FTD 确认（发生在反弹尝试 Day 4+）:
   - 上证综指或沪深 300 单日涨幅 ≥ 1.7%
   - 且成交量比前一日放大 ≥ 10%
   - → 市场方向从"修正"升级为"确认上行"
   - → 仓位可以从低仓提升到正常水平

3. 派发日（Distribution Day）:
   - 指数收盘下跌，且成交量比前一日放大
   - 在 4-5 周内出现 ≥ 5 个派发日 → 市场方向从"确认上行"降为"上行受压"
   - 在 4-5 周内出现 ≥ 6 个派发日 → 降为"市场修正"
   - 修正期间，禁止新买入，只持有或减仓
```

```
CAN SLIM 市场三态与本框架仓位联动:

  确认上行（Confirmed Uptrend）→ 体制不降级时，允许最高仓位
  上行受压（Uptrend Under Pressure）→ 总仓位强制 ≤ 40%，不新增
  市场修正（Market in Correction）→ 总仓位强制 ≤ 15%，不买入

  与现有体制联动:
    bear/range 体制 + FTD 确认上行 → 仍然保守（体制优先）
    bull 体制 + 市场修正 → 仓位打六折（CAN SLIM M 降级）
```

### 6.8 霍华德·马克斯周期定位（来源：《投资最重要的事》）

霍华德·马克斯的橡树资本以"周期定位"著称——不是预测方向，而是判断"我们在哪里"。他在《投资最重要的事》用一张钟摆图总结了市场心理：

```
           极度恐惧                        极度贪婪
           ←───────┬──────────────────→
                   │
    买入的最佳时机  │  卖出的最佳时机
    （现在？）      │  （何时？）
```

**当前周期定位四维打分**（基于可观察的市场指标，每月更新一次）：

```
维度1: 估值温度
  沪深300 PE 百分位 < 20%  → 偏低估（+1 分）
  沪深300 PE 百分位 20-40%  → 合理偏低（0 分）
  沪深300 PE 百分位 40-60%  → 合理（-0.5 分）
  沪深300 PE 百分位 > 60%  → 偏高估（-1 分）

维度2: 情绪温度
  新发基金规模 < 近3年均值50% → 极度悲观（+1 分）
  成交量/市值 < 近3年均值70%   → 偏冷淡（+0.5 分）
  成交量/市值 > 近3年均值150%  → 过热（-1 分）

维度3: 信用环境
  社融增速上升 + 信用利差收窄   → 宽松（+0.5 分）
  社融增速下降 + 信用利差扩大   → 紧缩（-1 分）

维度4: 政策信号
  近3个月有降准/降息/鼓励入市   → 友好（+0.5 分）
  近3个月有加准/加印花税/打压   → 不友好（-1 分）

周期总分 = 四维得分之和（范围 -4 到 +3）

  总分 ≥ +1.5  → 周期底部区域，马克斯建议"激进"（仓位 ×1.2）
  总分 0 ~ 1.5  → 中性区间
  总分 -1.5 ~ 0 → 偏谨慎（仓位 ×0.8）
  总分 < -1.5   → 周期顶部区域，马克斯建议"防守"（仓位 ×0.6）
```

> 马克斯："我们可能不知道会走向哪里，但我们应该知道现在站在哪里。"

## 七、卖出规则——10 条分层退出

### 7.1 为什么要卖出规则

格林布拉特在 Magic Formula 中的做法是：**每个季度末机械调仓**。芒格说："如果买的时候是对的，那就不应该卖"。但在 A 股场景下——T+1 结算、±10% 涨跌停、政策风险——你需要一套分层退出规则来保护本金。

卖出规则的层级：（越靠前越硬性，不可争议）

| 层级 | 类型 | 触发后必须执行 |
|------|------|-------------|
| SL（Stoploss） | 止损——保护本金 | 是，立即 |
| TP（Take Profit） | 止盈——保护利润 | 是，立即 |
| TE（Time Exit） | 时间退出——死钱/质量问题 | 是，但有执行窗口 |

### 7.2 现有 7 条规则（复用 portfolio/manager.py）

| 编号 | 类型 | 条件 | 适用周期 |
|------|------|------|---------|
| SL-1 | 硬止损 | 亏损 >= 止损% | 全部 |
| SL-2 | 移动止盈回撤 | 从持仓期间最高价回撤 >= 8% | 全部 |
| TP-1 | 目标止盈 | 盈利 >= 止盈% | 全部 |
| TP-2 | 快盈锁仓 | 持有 ≤ 3 天且涨幅 ≥ 8% | 短线 |
| TE-1 | 死钱退出 | 持有 ≥ 10 天，涨跌 ±2% 以内 | 中线 |
| TE-2 | 水下过久 | 持有 ≥ 5 天，亏损 ≥ 2% | 短线/中线 |
| OT-1 | 隔夜超时 | 时间 > 次日 10:15（隔夜策略专用） | 短线 |

### 7.3 新增 3 条规则（长线/中线专用）

| 编号 | 类型 | 条件 | 理论来源 | 适用周期 |
|------|------|------|---------|---------|
| **TE-3** | 质量恶化 | 新季报发布后 F-Score 下降 ≥ 2 分 | 皮奥特罗斯基：F-Score 恶化是公司基本面变差的最可靠信号 | 长线 |
| **TE-4** | 板块退潮 | 行业渗透率跌破 5% 或超过 40% | 费雪：产业过了加速期，继续持有的性价比大幅降低 | 中线/长线 |
| **TE-5** | 信号反转 | SUE 从正值转为负值且绝对值 > 1 | Bernard & Thomas：负 SUE 意味着盈利正在恶化 | 短线/中线 |

### 7.4 按时间周期差异化参数

| 周期 | 止损 | 止盈 | 最大持有天 | 移动止盈激活 | 移动止盈回撤 |
|------|------|------|----------|------------|------------|
| 短线（隔夜-5 天） | -3% | +8% | 5 | +5% | -4% |
| 中线（1-4 周） | -8% | +15% | 20 | +10% | -8% |
| 长线（1-6 月） | -15% | +30% | 120 | +15% | -12% |

**为什么要差异化？**

短线赚的是行为偏差的快速回归——退得慢了利润就没了。
长线赚的是企业价值的逐步实现——退得太快等于自己砍掉复利。
德雷曼在《逆向投资策略》中指出：过早止盈是个人投资者跑输机构的第一大原因。

### 7.5 费雪的三类卖出理由（来源：《怎样选择成长股》）

费雪是出了名的"不愿卖出"派。他说在 60 年的投资生涯中，值得卖出的情况只有三种：

| 费雪的卖出理由 | 量化触发条件 |
|-------------|------------|
| **1. 当初买入就错了** | 回头检查买入时的因子得分——如果当时综合得分 < 65，是错误买入，果断卖 |
| **2. 公司基本面发生永久性恶化** | F-Score 连续两季下降且总分 < 4，或 ROIC 从 > 15% 跌至 < 5% |
| **3. 发现了好得多的替代品** | 格林布拉特调仓日，替代品综合得分 > 当前持仓 + 10 分 |

```
费雪不卖的情形:
  - 股价跌了 20%？不卖——只要基本面没变，跌了更便宜
  - 大盘跌了 30%？不卖——这是买更多的机会，不是卖出理由
  - 持股两年了还不涨？不卖——费雪持有摩托罗拉 21 年才赚大钱
```

### 7.6 德雷曼恐慌买入规则（来源：《逆向投资策略》）

德雷曼的核心方法论：**在别人恐慌时贪婪**。他给出了具体的量化恐慌指标：

```
市场恐慌三信号（全部满足 → 额外买入机会）:

1. 全市场 PE 中位数处于近 5 年最低 10%
2. 近 5 个交易日有 ≥ 1 天出现"千股跌停"或跌停数 > 500
3. 媒体情绪极度负面（可用"恐慌/崩盘/股灾/暴跌"百度搜索指数飙升替代）

触发后动作:
  - 不追跌停股——等止跌后第二天再评估
  - 在 quality_pool 中优先选 F-Score ≥ 7 + ROIC ≥ 15% 的标的（德雷曼强调："在恐慌中买最优质的价值股，而不是捡最便宜的垃圾"）
  - 每次恐慌最多加仓 10% 总权益
```

> 德雷曼："在 1987 年黑色星期一之后买股票的人，后来都赚了大钱。在 2008-2009 年买股票的人也一样。问题是——你敢不敢？"

### 8.3 费雪 15 点选股清单（可量化子集，来源：《怎样选择成长股》）

费雪在 1958 年提出的 15 个问题，是基本面研究的经典框架。以下为可以量化的 9 条：

| # | 费雪问题 | 量化代理 | 在本框架的位置 |
|---|---------|---------|-------------|
| F1 | 产品/服务有足够的市场潜力，至少能保持几年的销售增长？ | 近 3 年营收 CAGR > 10% | G4（营收 > 1 亿） + F5 PEG |
| F2 | 管理层有决心持续开发新产品？ | 研发费用率 > 行业均值 | 需新增（A 股研发数据可用） |
| F3 | 研发努力相对于公司规模有效？ | 毛利率 > 行业均值（研发的成果体现） | F1 P8 毛利率上升 |
| F4 | 有高于平均水平的销售团队？ | 销售费用率的营收转化效率 | 需新增 |
| F5 | 有可观的利润率？ | 毛利率 > 行业均值 + 净利率 > 5% | F1 P8 |
| F6 | 做了什么来维持/改善利润率？ | 毛利率同比上升 + 费用率同比下降 | F1 P8 + P9 |
| F7 | 有出色的劳动和人事关系？ | （不可量化） | 跳过 |
| F8 | 有出色的行政关系？ | （不可量化） | 跳过 |
| F9 | 管理层有深度？ | CEO 任期 > 3 年 + 管理层持股 > 5% | 需新增 |
| F10 | 成本分析和会计控制好？ | 应计比率正常（F3） | F3 |
| F11 | 所在行业有独特的竞争优势？ | ROIC > 行业均值（护城河的经济证据） | F2 |
| F12 | 有短期/长期的利润前景？ | 分析师一致预期 EPS 增长 > 10% | 需新增 |
| F13 | 股权融资不会稀释股东？ | 无增发（F1 P7） | P7 |
| F14 | 管理层报喜也报忧？ | 近 3 年无信披违规 | G6 |
| F15 | 管理层有不可质疑的正直品格？ | 无财务造假 + 无大股东占款 | G6 |

```
费雪得分 = 满足条数 / 可量化条数（9 条）

≥ 7/9 → 优秀，标记为"费雪成长股"，F6 动量趋势分 +0.05
≤ 3/9 → 警告，标记为"非费雪型"，整体得分扣 0.05
```

### 8.4 芒格误判心理学避错清单（来源：《穷查理宝典》第 11 讲）

芒格在《穷查理宝典》最后一讲中，系统总结了 25 种人类误判心理倾向。以下是投资中最致命的 8 种及本系统的防护：

| # | 芒格列出的人类倾向 | 在选股中的体现 | 本系统防护 |
|---|-----------------|-------------|----------|
| M1 | **奖励/惩罚超级反应** | 买入后看到涨就开心，跌就痛苦——不是基于分析 | 11 因子打分强制理性，不凭感觉 |
| M2 | **喜爱/厌恶倾向** | 对持有一段时间的股票产生感情，忽略卖出信号 | 10 条机械卖出规则——触发就卖，不许讲借口 |
| M3 | **一致性与承诺倾向** | 公开说过"这股票好"后，即使变坏了也不敢反向操作 | 不做公开推荐——只有机器打分 |
| M4 | **过度乐观倾向** | 高估自己选股能力，把运气当实力 | 回测验证——只有数据能说话 |
| M5 | **被剥夺超级反应** | 亏损时不愿止损，希望"等它回来" | SL-1/SL-2 硬止损——不等 |
| M6 | **社会认同倾向** | 跟风买大家都在买的股票（A 股最常见的亏钱方式） | F9 渗透率 > 60% 的行业不碰——已经太拥挤了 |
| M7 | **锚定效应** | 锚定在买入成本价，以"是否回本"为标准评判 | 所有买卖基于当前因子评分而非成本价 |
| M8 | **确认偏见** | 买入后只找支持自己判断的信息 | 多因子来自不同学派——互相制衡 |

### 8.5 卡尼曼行为避错纪律（来源：Kahneman《思考，快与慢》）

卡尼曼区分了大脑的两个系统——系统 1（快，直觉）和系统 2（慢，理性）。投资亏损几乎全部来自让系统 1 干了系统 2 的活。

```
执行纪律铁律（每次买入前必须过一遍）:

1. 禁止模式识别幻觉:
   - "这个形态很像上次那支大牛股" → 这不是理由，查因子得分
   - 只看量化信号，不看图形相似性

2. 禁止可得性偏差:
   - "最近新闻全是这家公司" → 这不是买入理由
   - 只有 11 因子得分 ≥ 65 才触发买入

3. 禁止后见之明:
   - "我早知道会涨/跌" → 记录买入时的因子得分
   - 彼得·林奇方法：每次买入时写三行——"为什么买？预期多久？错了怎么出来？"

4. 禁止损失厌恶失控:
   - 止损触发后不找借口——格雷厄姆说"止损线是你欠市场先生的一笔债"
   - 米勒维尼说"卖错了可以再买回来——但前提是你卖了"

5. 禁止过度交易诱惑:
   - 没有信号就不动——格林布拉特的机械调仓思想
   - 费雪说"如果你做对了买入决策，卖出时机很少到来"
```

### 9.5 奥肖内西反过拟合原则（来源：《华尔街股市投资经典》第 21 章）

奥肖内西在分析 50+ 因子后发现：**绝大多数"神奇策略"都是数据挖掘的幻觉**——在回溯测试中表现惊人，在未来完全失效。他给出了四条防过拟合铁律：

```
铁律 1: 因子必须跨周期有效
  → 如果一个策略在 1963-1979 和 1980-2007 两个子区间的表现相差 > 50%，则大概率是过拟合

铁律 2: 因子必须有经济学解释
  → 不能只因"回测好看"就纳入——每个因子都必须有合理的行为/经济解释
  → 本框架所有 11 个因子都有经典文献支撑（见第 3.5 节）

铁律 3: 参数不能过度优化
  → 阈值用"逻辑推导"而非"回测最优值"
  → 例：F-Score ≥ 6 不是因为回测显示 6 最好，而是因为 ≥ 6 意味着超过一半的指标在改善
  → 例：PEG < 1.0 不是回测最优值，而是林奇的理论推导

铁律 4: 因子组合必须精简
  → "更多的因子"不等于"更好的策略"
  → 奥肖内西发现：3-5 个互不相关的因子组合往往优于 10+ 个因子的大杂烩
  → 本框架 11 因子分四层，但每层内部因子低相关（质量 ≠ 价值 ≠ 动量 ≠ 信号）
  → 如果发现两个因子高度相关（r > 0.5），合并或删除一个
```

> 奥肖内西："如果你有一把 50 个因子的猎枪，你总能找到在历史数据中打中靶心的那个因子。问题是——那个靶心在未来还会在同一个位置吗？"

## 十、执行纪律：芒格 20 孔打孔卡 + 林奇买前三行（来源：《穷查理宝典》《战胜华尔街》）

选股体系再好，执行的人控制不住自己是白费。两本书提供了最精简的执行框架：

### 芒格 20 孔打孔卡

> "如果我管理一个大学捐赠基金，我会给投资委员会每人发一张 20 孔的卡。每做一笔投资，就打一个孔。20 个孔都打完，这辈子不能再做任何投资。"—— 芒格

```
规则: 每人每年最多 10 笔交易（买入 + 卖出各算一笔）
  → 6 万本金，一年 10 次买卖，每次对应 5-10% 总权益
  → 如果 10 个孔都打完了？停手，不再交易，等明年
  → 这个规则强制你想三遍再动手
```

### 林奇买前三行

林奇在富达基金的办公桌上放着一本"买前三行"笔记本——每笔交易前手写三行：

```
每次买入前，在笔记本上手写:

第 1 行: 为什么买这只？（引用至少 1 个因子的量化数据，不能写"感觉会涨"）
第 2 行: 预期持有多久？（选一个：3 天 / 3 周 / 3 个月）
第 3 行: 如果错了，什么时候认错？（指定一个具体的退出条件）

林奇说：
  "如果你写不出这三行，就说明你根本不知道自己在干什么。
   这种情况下，去买指数基金，别买个股。"
```

## 十二、A 股特殊约束处理

| 约束 | 影响 | 处理方式 |
|------|------|---------|
| T+1 结算 | 买入当日不能卖，隔夜有风险 | 短线仓位买入前过 TE/OT/体制检查三层风控 |
| ±10% 涨跌停（主板） | 涨停买不到，跌停卖不掉 | 涨停当日不追买（FUNDAMENTAL_RULES 第 1 条） |
| 6 万本金 | 佣金侵蚀不可忽视 | 单只最小 0.3 万（佣金占比 < 0.5%），月换手 < 50% |
| 最多 5 只持仓 | 集中度天然高 | 漏斗体系本身就是集中化的，5 只已足够 |
| 仅主板（60/00） | 标的池缩小 ~40% | 排除 300/688/8 开头，主板仍有 ~3000 只足够筛选 |
| 季报发布滞后 | F-Score 更新不及时 | 增量采集 + 公告日匹配，超过 45 天标记 "stale" 降权 |
