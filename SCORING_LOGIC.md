## 一、架构概览

```
research_data/<date>/intraday/   → 盘中快照
data/<date>/fund_flow.json       → 收盘数据
portfolio/black_swan.py          → 黑天鹅检测
                                  │
                                  ▼
                    score_all_stocks(date_str)
                                  │
                    ┌─────────────┼─────────────┐
                    │             │             │
              体制检测+黑天鹅   23因子评分    P因子修正
              bull/bear/range   加权求和      ±0.15微调
                    │             │             │
                    └─────────────┴─────────────┘
                                  │
                    ┌─────────────┼─────────────┐
                    │             │             │
              scores.csv      sector_scores  买入引擎
              综合/短线/中线/启动  板块评分    三级过滤
```

## 四、策略周期分离

| 得分类型 | 核心因子 | 用途 |
|------|------|------|
| **综合得分** | 23因子加权 | 通用排名 |

| **启动得分** | 低位+启动因子提权 | 底部抄底 |
