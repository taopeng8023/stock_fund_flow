# K线形态胜率发现报告

**日期**: 2026-07-02  
**数据源**: `baostock_data/data/20260702/daily/` — 3,800+ 只 A 股日 K 线  
**引擎**: `kline_discovery.py` v2 — 确认日机制 + 多条件 AND 组合  
**目标**: 发现胜率 ≥85% 的 K 线交易策略  

---

## 方法论

### 信号流程

```
形态触发日 T → T+1 确认日 → T+1 收盘入场 → 持仓 N 日 → 计算收益
```

### 确认级别

| 级别 | 标记 | 条件 |
|------|------|------|
| Standard | (无) | T+1 收盘 > T 收盘 |
| Ultra-Strict | ▲ | T+1 收盘 > T 最高 + T+1 收阳 + T+1 放量 |
| Bull Regime | ★ | ▲ + MA20 上升 + 价在 MA20 上 |
| Hyper-Strict | ⬡ | ▲ + MA20 上升 + 低波动 + MA5>MA10 + 量增 >1.2x |
| 双形态叠加 | ◆ | 两模式 3 日内同时触发 + ▲ 确认 |

### 参数

- 最小 K 线数: 80 根
- 盈利阈值: 0.5%（过滤噪音）
- 持仓周期: 1, 2, 3, 5, 10, 15 日
- 采样: 50 → 2000 只随机股票，分批次递增
- 形态库: 9 大类 36 个形态 + 6 个强化形态 (J 系列)

## 关键发现

### 确认级别对胜率的影响

| 确认级别 | 平均胜率提升 | 样本衰减 |
|----------|-------------|---------|
| Standard → ▲ | +8-15% | -40-60% |
| ▲ → ★ | +3-8% | -30-50% |
| ▲ → ⬡ | +5-10% | -60-80% |

- **▲ (Ultra-Strict)** 是性价比最高的确认级别 — 胜率提升显著且样本充足
- **★ (Bull Regime)** 进一步过滤但样本下降明显
- **⬡ (Hyper-Strict)** 胜率最高但样本过少，实用性有限

### 最稳健的形态类型

1. **趋势回调类** (B1, B4, B5): 均收益高，样本适中，适合中线
2. **反转反包类** (D1, D2): 胜率最高，适合超短线
3. **超跌反弹类** (A3): 均收益极高但信号稀少

### 样本量注意事项

- n ≥ 20 的形态胜率通常回落至 55-65%
- n ≥ 15 是统计显著性的最低要求
- n ≥ 8 的高胜率形态适合作为高置信度信号叠加使用
