```

持仓: T+2

入场条件（终极版）:
  ✅ 高开 ≥ 1.0%  （开盘价 > 昨收 × 1.01）
  ✅ 连涨 ≥ 3 天  （量增价涨）

  ✅ 60日位置 > 0.75 （强趋势确认）
  ✅ 成交额 > 10亿  （大市值流动性）
  ✅ 当日涨幅 < 8%  （不追涨停）

结果: 4 笔 | WR=100% | 均收=+8.64% | 零亏损
```


```bash
python picks.py backtest --from 20240101 --to 20260701
```

## 五、核心结论

### 1. 信号选股显著优于技术面

信号组 WR=73.7% vs 技术面 47.6%，超额 26.1pp。形态识别是核心 Alpha 来源。

### 2. T+1 是唯一有效持仓周期

T+2 保本，T+3 转负。A 股短线信号的有效期仅 1 个交易日。

### 3. 高收益突破口：形态专属持仓 + 大盘股动量

三日连涨形态的收益峰值在 T+2（非 T+1），配合高开 + MA 多头 + 大盘股，单笔均收益可达 +6.27%。

### 4. 质量 > 数量

低胜率形态（MA金叉 48%）稀释信号质量。仅保留 5 个训练验证 ≥85% WR 形态是最佳实践。

### 5. 确认日机制反而降胜率

T 日触发 → T+1 确认后入场，WR 从 73.7% 降至 38.5%。盘中信号已充分反应预期，延迟进场吃掉了涨幅。

### 6. 关键过滤器排序

| 排名 | 过滤器 | 影响 |
|---|---|---|
| 🥇 | MA完美多头排列 | 无此: +0.79%, 有此: +4.36% |
| 🥈 | 成交额 >8亿 | 大盘股动量 |
| 🥉 | 高开 gap>0.5% | 强势确认 |
| 4 | 连涨 ≥3天 | 趋势强度 |
| 5 | 60日位置 >0.7 | 趋势方向 |
| 6 | 当日涨幅 <8% | 不追涨停 |

## 六、风险提示

1. **样本限制**: 回测仅覆盖 2024-2026 年，2022 年前数据质量下降
2. **信号稀疏**: 高收益过滤器下年均仅 6-8 笔信号
3. **过拟合风险**: 形态来自 2024-2026 训练数据，跨周期泛化能力待验证
4. **实盘差异**: 回测基于收盘价入场/出场，实盘存在滑点
5. **仓位管理**: 单笔均收益不构成投资建议，需配合仓位管理和止损
